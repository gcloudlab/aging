<h1 align="center">Aging</h1>

## Feature

- 用一张照片推演你从年幼到年老的 3s Gif
- 使用 Workers 从 Cloudflare R2 存储和检索照片
- 通过 Upstash 缓存24小时后自动删除照片
- NextAuth 授权，MongoDB 存储用户信息
